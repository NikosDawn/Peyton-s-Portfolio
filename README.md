# Peyton's Portfolio

## Project Description

This project is a multi-page personal portfolio website created as part of my web development coursework. The site showcases my background, skills, projects, and career goals while demonstrating semantic HTML, responsive CSS, accessibility best practices, and version control using Git and GitHub.

## Pages

* Home – Introduction and overview of my portfolio.
* About – Information about me, my skills, and my interests.
* Projects – A collection of featured projects and coursework.
* Resume – My education, experience, career goals, and contact information.

## Technologies Used

* HTML5
* CSS3
* Git
* GitHub
* Visual Studio Code
